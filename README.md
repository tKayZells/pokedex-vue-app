# Pokedex Vue App

A web app built using Vuejs and [PokeAPI](https://github.com/PokeAPI/pokeapi) with HQ images by [HybridShivam](https://github.com/HybridShivam/pokemon/)

## Project setup
```
npm install
npm run serve
```
